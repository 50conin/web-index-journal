# web-index-journal

本仓库用于归档与发布多个独立的 HTML 页面。这些页面不针对任何特定域名或网站，仅作为静态网页的索引与展示用途。

## 项目简介

web-index-journal 是一个轻量级的 HTML 页面集合，旨在提供一种简洁的方式保存和分享各类独立网页内容。仓库中的页面彼此独立，不依赖外部数据源或动态服务，可直接在浏览器中打开查看。

## 目录说明

```
.
├── index.html          # 仓库入口页面，列出所有归档页面的链接
├── pages/              # 存放各个独立 HTML 页面的目录
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 共用资源文件（CSS、图片等，可选）
└── README.md           # 本文件
```

- **index.html**：仓库的主索引页，方便快速浏览所有归档页面。
- **pages/**：每个独立 HTML 页面均放置于此目录下，命名规则为 `page-xxx.html`。
- **assets/**：存放全局共用的样式、图片或脚本文件，非必需。

## 页面归档说明

归档页面为完整的 HTML 文件，包含独立的结构、样式与内容。所有页面均遵循以下原则：

- 不包含外部链接或跳转至其他站点的功能。
- 不使用动态脚本（如 JavaScript 异步请求）。
- 内容保持静态，不随时间或用户交互改变。

这些页面可视为网页快照或信息卡片，适合长期保存与离线查看。

## 维护说明

- 新增页面时，请确保文件命名不与现有重复，并更新 `index.html` 中的链接。
- 如需删除或修改页面，建议同步更新索引文件以保持一致性。
- 不鼓励在页面中嵌入外部资源（如 CDN 文件），以免影响离线可用性。
- 本仓库不追踪页面来源或历史，仅关注内容本身。
